# Raspberry Pi Pico Entertainment System — “Arcadium”

**Embedded Systems & Hardware Prototyping • Team Engineering Project**

A modular entertainment system integrating **Raspberry Pi Pico**, LEDs, LCD screens, and physical controls in a custom enclosure.

### Key Contributions
- Developed game logic prototypes, wiring diagrams, pseudocode
- Built laser-cut and 3D-printed enclosure
- Maintained full engineering notebook documenting design iterations
- Designed modular hardware workflow for multiple game expansions
